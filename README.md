# LuminaCalc - Minimalist Gradient Calculator

## How to Run
This is a standard web application. You can verify it immediately by:
1.  **Double-clicking** the `index.html` file in this folder.
2.  Right-clicking `index.html` and choosing "Open with Google Chrome" (or Edge/Firefox).

## Project Structure
- `index.html`: The main structure and layout.
- `style.css`: Contains all the styling, gradients, and glassmorphism effects.
- `script.js`: Handles the calculator logic, math functions, and solvers.

## Features
- **Tabs**: Switch between Normal, Algebra, Trigonometry, and Calculus modes.
- **Solvers**: Scroll to the "Solvers" tab to find Linear, Quadratic, and Matrix tools.
